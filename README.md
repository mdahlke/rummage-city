# Rummage City

## Requirements
PHP >=7.2  
Composer  
Node  


## Install

Clone the repo  
```shell script
git clone git@github.com:mdahlke/Rummage-City.git rummagecity.dev
```

Install dependencies
```shell script
composer install
```
```shell script
npm install
```

Link Storage
```shell script
php artisan storage:link
```

Run migrations
```shell script
php artisan migrate
```


### Development
```shell script
npm run watch
```

### Testing
```shell script
composer test
```
